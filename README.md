# CLI chat with GO websocket

<img src="assets/logo.jpg" style="width: 100%"/>

This is a simple chat CLI written in Go


Steps to start the server

```
cd ./src
go get github.com/gorilla/websocket
go run main.go
```

Steps to start CLI

```
pip3 install -r requirements.txt
python3 clichat.py
```
